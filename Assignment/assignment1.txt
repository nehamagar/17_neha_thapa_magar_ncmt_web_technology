 9  A user types www.google.com into their browser, but the browser displays a “Server Not Found” error even though the internet condition is active. However, accessing the site via 142.250.190.46 works. Diagnose the problem and explain the underlying technology that failed. 
-The problem is a DNS (Domain Name System) failure. 
-when the user types www.google.com, the browser must translate the domain name into an Ip address using DNS. 
-Since accessing the site directly via the address, it shows that the internet connection and web server are functioning correctly. 
-However, the DNS server failed to resolve the domain name into its IP address and it caused the failure.
